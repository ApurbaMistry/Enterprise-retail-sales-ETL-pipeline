# Enterprise Retail Sales ETL Pipeline

## Overview

This project demonstrates a complete ETL (Extract, Transform, Load) pipeline built using Python, Pandas, NumPy, SQLite, SQL, and Logging.

- Pandas
- NumPy

- SQL
